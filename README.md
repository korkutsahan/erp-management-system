A desktop **ERP (Enterprise Resource Planning)** system designed to help companies manage business operations from a single platform.

---

## Project Overview

This project aims to develop a centralized ERP system that allows companies to manage financial and operational processes efficiently.

The system enables tracking of:

- Income and expenses
- Stock management
- Customer records
- Supplier records
- Debt and receivable tracking
