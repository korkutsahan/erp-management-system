# ERP Management System

This project is a desktop ERP (Enterprise Resource Planning) system designed to help companies manage operational and financial processes from a single platform.

## Features

- Income and expense tracking
- Debt and receivable tracking
- Stock management
- Customer management
- Supplier management

## Technologies

- Python
- Qt Designer
- SQL

## Project Status

This project is currently under development.  
The UI design phase has been completed, and the coding process is planned as a team project.
